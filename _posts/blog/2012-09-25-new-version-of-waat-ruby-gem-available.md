I finally got around to pushing out a new version of [**WAAT-ruby (1.4.1)**](<https://rubygems.org/gems/WAAT>) on rubygems.org. See the [**WAAT-ruby project on github**](<https://github.com/anandbagmar/WAAT-ruby>) for more information.  
  
Watch this space for a new version of WAAT-ruby that overcomes the limitation of doing web analytics automation for https urls.
