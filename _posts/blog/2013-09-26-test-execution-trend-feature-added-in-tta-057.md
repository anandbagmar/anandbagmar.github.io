A new version of [**_Test Trend Analyzer (TTA)_**](<http://essenceoftesting.blogspot.in/search/label/tta>)  is now available on [github](<https://github.com/anandbagmar/tta>). 

  


These are the new features in it:

  


  * **Test Execution Trend**  \- to see the benchmarking of specific test execution over a time period



[![](https://blogger.googleusercontent.com/img/b/R29vZ2xl/AVvXsEigfYnddr4aVCEsTPj8w5RRPDOLvCURKGJakVXlRe6RwGmkHy9F6SUfJZIiRVd7oXF5TRfHSCPSH5lhbftc-Ufo5wXQ_jYwRyymtdq1zZfypj4HbnhdEwRw9zrPUcMcBXzRAy3ZMatYrzzL/s200/18.+TestExecutionTrend.png)](<https://blogger.googleusercontent.com/img/b/R29vZ2xl/AVvXsEigfYnddr4aVCEsTPj8w5RRPDOLvCURKGJakVXlRe6RwGmkHy9F6SUfJZIiRVd7oXF5TRfHSCPSH5lhbftc-Ufo5wXQ_jYwRyymtdq1zZfypj4HbnhdEwRw9zrPUcMcBXzRAy3ZMatYrzzL/s1600/18.+TestExecutionTrend.png>)

  * **Failure Analysis View**


    * Modified view to show failures for specific Test category along with specific test run


    * Modified view to show failures for All test categories with latest test run


  * Minor Refactoring for **Compare Runs**  view


  * Query optimisations + minor defect fixes.

For those that missed the earlier announcement - summary of features of TTA is available [here](<http://essenceoftesting.blogspot.in/search/label/tta>), and source code is available on [github](<https://github.com/anandbagmar/tta>).

  


  


PS: There is a lot of work (features / UI) planned for TTA. If you wish to contribute, we would love your help!!
