The newest version of WAAT (Web Analytics Automation Testing framework) for [**Java**](<https://github.com/anandbagmar/WAAT>) and [**Ruby**](<https://github.com/anandbagmar/WAAT-ruby>) is now available with a new plugin - JsSniffer.  
  
Following the [announcement](<http://essenceoftesting.blogspot.in/2012/10/waat-java-ruby-with-jssniffer-is-out-of.html>) of the new JsSniffer plugin for WAAT, I have now completed the testing for the same, and released the gem - **[WAAT-1.5.0.gem](<http://rubygems.org/gems/WAAT>)** to rubygems.org.  
  
**_Please take a look at the  _**[FAQs](<https://github.com/anandbagmar/WAAT/wiki/FAQs>) and the **[major changes  on Ruby](<https://github.com/anandbagmar/WAAT-ruby#changes-in-v150>) / [major changes on Java section](<https://github.com/anandbagmar/WAAT#changes-in-v150>)**** _ _** for some known issues, potential solutions for the same _\- as**some of these changes could break your tests.**_
